记一次 Electron 遇到的 bug

date_range 2018-10-26
local_offer electron

最近和老板谈了个客户端开发的项目，要用到 electron ，刚好项目比较闲，先抽空搭个框架。

由于之前做小项目用过 electron-vue 这套脚手架，感觉还是蛮不错的，于是分分钟搭了起来，先搞了一点ipc通讯的组件，在使用 vuex 的时候发现分发不了action,我以为自己哪里写的有问题，检查了一遍，也就那么几行代码，没啥问题啊。

然后又怀疑是不是依赖包没装好或者 node 版本不对，又花了好长时间重装了一遍，依然没用。

实在不行又重启电脑，还不管用，那是不是这个版本出现问题了呢？

我翻出之前的老项目启动起来测试了一下，发现功能正常，基本上可以确定是新版的问题了。但是要装回老版本，我肯定是不愿意的，不如先搞搞别的，看看官方会不会修复这 bug ，毕竟这项目还没谈成，我只是闲着做一点预备工作而已。

于是先做一点用不到 vuex 的部分，继续和老板推动这个项目的谈判。没过一两天就发现有个国内的哥们操着Google翻译的英文给这个仓库作者提出了这个 bug ，看来我的判断没错，接下来就等解决方案了，结果是解决了这个 bug ，也把环境整合的差不多了，这个项目却没谈成，最终烂尾收场。

虽然项目没做成，但在解决问题的过程中大概了解了这个库的一些结构，也是很有意思，目录组织很不错，开发体验上感觉跟做 vue 的 web 项目没区别。
